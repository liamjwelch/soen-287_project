To setup the database:

1. install mysql
2. setup a user with the name of your local user and the password `password` on your local mysql instance
3. create a database named `soen287` on the mysql instance.
4. grant all privileges on the database `soen287` to the user created in step 2
5. start the web server and open the page /database/init.php in your browser.
6. This will create all required mysql tables
7. You can also use this page to reset a specific table or all tables.
